## CLI — 检查

```bash title="audit-cli.sh"
carina sessions
carina audit SESSION
carina audit verify SESSION
carina audit last
carina report SESSION
carina export SESSION
carina items SESSION
```

## CLI — 补丁

```bash title="patch-cli.sh"
carina patch list SESSION
carina patch show SESSION PATCH_ID
carina patch propose SESSION PATH
carina patch apply SESSION PATCH_ID
carina patch rollback SESSION PATCH_ID
```

## 配方：错误编辑之后

<Steps>
  <Step title="找到会话">
    ```bash frame="none"
    carina sessions
    carina audit last
    ```
  </Step>
  <Step title="列出补丁">
    ```bash frame="none"
    carina patch list SESSION
    ```
  </Step>
  <Step title="回滚">
    ```bash frame="none"
    carina patch rollback SESSION PATCH_ID
    ```
  </Step>
  <Step title="再验链">
    ```bash frame="none"
    carina audit verify SESSION
    ```
  </Step>
</Steps>

## 不是 Git 替代品

审计与补丁解释**会话内** Agent 行为；不替代 Git 历史与 PR 审查。
