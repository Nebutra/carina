Carina 是**本地优先的 AI Agent 运行时**：在编程 Agent 从对话转向真实仓库操作时，提供策略门控、哈希链审计与事务回滚。

## 一句话

> Carina 是让编程 Agent 在本机全力执行的本地运行时——每次动作都经过策略，写入哈希链审计，并可回滚。

## 价值表

| 需求 | Carina |
| --- | --- |
| 在仓库上行动 | 会话、任务、模型适配 |
| 控制副作用 | 能力内核、配置、审批 |
| 解释发生了什么 | 哈希链审计、item stream |
| 从坏编辑恢复 | 事务补丁回滚 |
| 嵌入其他产品 | JSON-RPC、Gateway、MCP、SDK |

```
Client ──JSON-RPC──▶ Go Control Plane ──▶ Rust Kernel ──▶ Zig Tools
```
