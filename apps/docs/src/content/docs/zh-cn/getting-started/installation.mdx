Carina 以本地工具链形式发布：CLI、常驻 daemon、可选 worker，以及 TUI / IDE / Web 等客户端表面。

## 环境要求

- macOS 13+ 或现代 Linux（x86_64 / arm64）
- 仅在拉取模型或包时需要网络（运行时本身 local-first）

## 安装

任选一种渠道。需要完整工具链时优先使用 Shell 安装脚本。

<Tabs>
  <Tab label="Shell">
    ```bash title="install.sh" frame="none"
    curl -fsSL https://carina.nebutra.com/install.sh | sh
    ```
  </Tab>
  <Tab label="Homebrew">
    ```bash title="Brewfile" frame="none"
    brew install nebutra/tap/carina
    ```
  </Tab>
  <Tab label="npm">
    ```bash title="package.json" frame="none"
    npm install -g @nebutra/carina
    ```
  </Tab>
  <Tab label="源码">
    ```bash title="install-from-source.sh"
    git clone https://github.com/nebutra/carina.git
    cd carina
    make install
    ```
  </Tab>
</Tabs>

## 验证

```bash title="verify.sh"
carina version
carina doctor
```

`carina doctor` 会检查 daemon 可达性、配置路径与常见误配置。

<Callout variant="note">
  包名与安装 URL 可能随开源发布渠道变化。升级后请优先运行 `carina doctor`。
</Callout>
