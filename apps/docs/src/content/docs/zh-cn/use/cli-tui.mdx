CLI / TUI 是本机操作 Carina 最快的方式。与 JSON-RPC 同一运行时 — 不同表面。

## 常用命令

| 命令 | 用途 |
| --- | --- |
| `carina daemon start` | 启动本地 daemon |
| `carina doctor` | 健康检查（套接字、内核、工具） |
| `carina` / `carina run "…"` | 交互 TUI 或一次性任务 |
| `carina audit SESSION` | 回放决策与副作用 |
| `carina audit verify SESSION` | 哈希链完整性 |
| `carina patch list \| rollback` | 列出 / 回滚事务补丁 |
| `carina call METHOD` | 脚本化 JSON-RPC |
| `carina workflow run NAME` | 跑工作流 DAG |

## 典型一天

<Steps>
  <Step title="拉起 daemon">
    ```bash frame="none"
    carina daemon start && carina doctor
    ```
  </Step>
  <Step title="在仓库里干活">
    ```bash frame="none"
    cd /path/to/repo
    carina run "your task"
    # 或: carina
    ```
  </Step>
  <Step title="检查 / 恢复">
    ```bash frame="none"
    carina audit SESSION
    carina patch list SESSION
    ```
  </Step>
</Steps>
