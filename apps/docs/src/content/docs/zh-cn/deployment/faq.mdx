## Carina 是托管云 Agent 吗？

不是。Carina **本地优先**。云身份/同步可以包裹它，但策略与工具执行仍属本地权威。

## Carina 会取代 Git 吗？

不会。补丁回滚恢复会话内 Agent 编辑；Git 仍是协作与长期历史的事实来源。

## 可以完全离线吗？

运行时本身可以。模型提供商与包安装可能仍需网络。
