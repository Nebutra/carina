# Third-party brand marks

These marks identify explicit outbound actions and are not part of Carina's
visual identity.

- `openai-blossom.svg`: the monochrome Blossom geometry from OpenAI's official
  [Design Guidelines](https://openai.com/brand/), cropped to the supplied mark
  bounds without altering its path or proportions.
- `claude-icon.svg`: `ClaudeIcon-Rounded.svg` from Anthropic's official
  [press kit](https://www.anthropic.com/press-kit), unchanged apart from
  non-semantic SVG definition IDs.

Keep both marks visually subordinate to the Carina wordmark and use them only
next to links that open the corresponding service.
